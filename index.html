xeq160.zip

XEQ - COM File Library and Command Executor
This program allows COM files to be stored
inside XEQ's code and executed by the
command: XEQ command
For example, if BEEP.COM has been added to
XEQ then: XEQ BEEP will execute the BEEP
program stored inside XEQ.
The original BEEP.COM can be removed from the
hard disk (onto a backup floppy!!!).
If the specified command is not in XEQ it is
passed to the DOS for execution as if
directly typed at the DOS prompt.
The purpose is to prevent wasted hard-disk
space by those pesky little COM files we
don't seem to be able to do without.
When hard disks are formatted they are given
an allocation size (called a cluster size)
and this is typically 2048 or 4096 bytes.
So when you save a 7 byte COM file like
BEEP.COM, it takes up a full cluster.
By combining these COM files into this
program or LIBRARY, several COM files can be
stored in one file (the XEQ program itself)
and still be accessed without this waste in
hard disk space.
